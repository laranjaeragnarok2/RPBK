# **App Name**: Nightlife Pulse

## Core Features:

- Hero slider: Event Promotion: Showcasing event details with a dark mode theme, replicating the 'Nightlife demo' aesthetic.
- Real-time Countdown: Countdown timer for featured events, enhancing user engagement and anticipation.
- Social Feed Integration: Embed social media feeds in real-time, providing users with the latest updates.
- Image Gallery: Display images of events.

## Style Guidelines:

- Primary color: Vibrant cyan (#B6FA0C) for an energetic and modern feel against the dark background.
- Background color: Dark charcoal (#232327) to maintain the 'Dark Mode' theme.
- Accent color: Electric purple (#405119) for highlighting interactive elements.
- Headline font: 'Poppins' (sans-serif) for a clean, modern title, especially in shorter texts. Body text: 'PT Sans' (sans-serif) to support long-form reading. Use 'Poppins' for headlines and 'PT Sans' for the rest.
- Use glowing neon icons to enhance the nightlife theme.
- Subtle parallax scrolling effects on background images to enhance the immersive experience.
